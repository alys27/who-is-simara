# Şəkillər Qovluğu

Bu qovluğa aşağıdakı şəkilləri əlavə etməlisiniz:

## Tələb olunan şəkillər:

1. **profile.jpg** - Profil şəkliniz (250x250px və ya kvadrat format, ən azı 500x500px)
   - Format: JPG, PNG
   - Ölçü: 250x250px (veb-saytda istifadə üçün)
   - Məsləhət: Yüksək keyfiyyətli, aydın şəkil

2. **project1.jpg** - Layihə 1 üçün şəkil (200px hündürlük, eni avtomatik)
   - Format: JPG, PNG
   - Ölçü: ən azı 800x600px (veb-saytda 200px hündürlükdə göstəriləcək)
   - Məsləhət: Layihənin ekran görüntüsü və ya fotoşəkli

3. **project2.jpg** - Layihə 2 (Hour of Code) üçün şəkil (opsional)
   - Format: JPG, PNG
   - Ölçü: ən azı 800x600px

4. **project3.jpg** - Digər layihələr üçün şəkil (opsional)
   - Format: JPG, PNG
   - Ölçü: ən azı 800x600px

## Şəkil əlavə etmək:

1. Şəkillərinizi bu qovluğa kopyalayın
2. Fayl adlarını yuxarıdakı adlarla eyniləşdirin (məsələn: `profile.jpg`)
3. Şəkillərin keyfiyyətini yoxlayın

## Placeholder şəkillər:

Əgər hələ şəkilləriniz yoxdursa, aşağıdakı onlayn xidmətlərdən istifadə edə bilərsiniz:
- https://placeholder.com/ - Placeholder şəkillər
- https://unsplash.com/ - Pulsuz yüksək keyfiyyətli şəkillər
- https://pixabay.com/ - Pulsuz şəkillər

## Qeyd:

Şəkillərin ölçüsünü kiçiltmək üçün onlayn alətlərdən istifadə edə bilərsiniz:

- https://www.iloveimg.com/resize-image - Şəkil ölçüləndirmə
